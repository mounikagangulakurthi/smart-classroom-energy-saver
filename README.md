# Smart Classroom Energy Saver

## Project Overview
This project uses ESP32, LDR Sensor, and PIR Motion Sensor to automatically control classroom lighting based on ambient light intensity and human presence.

## Components Used
- ESP32
- LDR Sensor
- PIR Motion Sensor
- LED
- Resistor

## Working Principle
The LED turns ON only when:
- The classroom is dark
- Motion is detected

Otherwise, the LED remains OFF.

## Features
- Automatic Lighting Control
- Energy Saving
- Real-Time Monitoring
- Low-Cost Implementation

## Applications
- Smart Classrooms
- Libraries
- Offices
- Smart Homes
